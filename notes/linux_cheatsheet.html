Zsh & Terminal Cheatsheet (macOS)

Do Básico ao Avançado (80/20) • Shortcuts • Flags Reference

1. Conceitos Fundamentais

/ Raiz do sistema
~ Home ($HOME)
. Diretório atual
! Case Sensitive: 'File' ≠ 'file'

2. Navegação & Arquivos

ls -l Lista detalhada
cd [dir] Mudar diretório
pwd Caminho atual
mv A B Mover/Renomear
cp A B Copiar A para B
rm [arq] Apagar (sem volta)
mkdir [d] Criar pasta
clear Limpar tela

3. Leitura & Texto

cat [arq] Ler tudo
less [arq] Ler paginado ('q' sair)
grep "txt" Buscar texto em arq
wc -l Contar linhas

4. Sistema & Processos

top Monitor CPU/RAM ('q')
df -h Espaço em disco
killall [x] Matar programa
last -1 Último login
date / cal Data / Calendário

5. Rede & Conectividade

curl -O [url] Baixar arquivo
ipconfig getifaddr en0 IP Local
whois [site] Info domínio
rsync -aE Sincronizar pastas

6. Avançado & Pipes

| Pipe (Saída vira entrada)
> Redirecionar p/ arq
* Curinga (Todos arqs)
sudo [cmd] Como Admin (Root)
man [cmd] Manual de ajuda

7. Atalhos Zsh (Edição)

Controle

Ctrl + C Interromper
Ctrl + D Sair (EOF)
Ctrl + L Limpar tela
Ctrl + Z Suspender

Navegação (Linha)

Ctrl + A Início da linha
Ctrl + E Fim da linha
Alt + B Volta palavra
Alt + F Avança palavra

Edição

Ctrl + U Apaga linha inteira
Ctrl + K Apaga cursor → fim
Ctrl + W Apaga palavra ant.
Alt + D Apaga palavra próx.
Ctrl + Y Cola (Yank)

Histórico

Ctrl + R Busca reversa
!! Repete anterior
!str Último c/ 'str'

8. Nano Editor

Ctrl + O Salvar
Ctrl + X Sair
Ctrl + W Buscar
Ctrl + ^ Selec. Texto
Alt + ^ Copiar
Ctrl + U Colar

Long Flags

10. macOS Extras

Cmd + C Copiar
Cmd + V Colar

Cmd + T Nova aba

11. Jobs

bg Mandar p/ background
fg Trazer p/ foreground
jobs Listar jobs
kill %1 Matar job 1
